# libpythonbr
Explicação de Criação de Projeto

Suportado Python 3


Para instalar:

``` console
python -m venv venv
venv/Scripts/activate
pip install -r requirements-dev.txt
```

Para conferir a qualidade do código:

``` console
flake8
```
